### Word Filter
Word Filter: A tool which uses keywords matching and Machine Learning for text filtering

#### Feature
- HTTP API
- Keywords Matching
- Model to predict

#### Usage
- Build executable: `go build`
- Run: `./word_filter `
- Run background: `./run.sh start`
- Usage: `./run.sh`
